# opendsr-std/homebrew-tap

Homebrew formulae for [opendsr](https://github.com/opendsr-std) tools.

## Install

```bash
brew tap opendsr-std/tap
brew install seedfaker
```

## Available formulae

| Formula     | Description                            |
|-------------|----------------------------------------|
| `seedfaker` | Deterministic synthetic data generator |

## Updating

```bash
brew update
brew upgrade seedfaker
```
